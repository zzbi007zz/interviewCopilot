# Design Guidelines

## UI Intent

Provide a compact floating assistant panel for live interview support.

## UX Rules

- Keep panel concise and low-noise.
- Prioritize readability under pressure.
- Present answer options as short bullets.
- Make loading/error/success states explicit.

## Layout Baseline

- Target compact width around 300-320px.
- Keep controls minimal (status, settings, close).
- Preserve high contrast and keyboard accessibility.

## Localization

- Match response language to detected interviewer language (EN/VI).
